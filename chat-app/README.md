# Chat-App

Chat-App is a basic messaging application that includes real-time chat functionality along with AI-powered chat features. This project is being reuploaded to this GitHub repository as part of my portfolio, as I have lost access to my previous account. The original repository can be found at [https://github.com/Devansh2835/chat-app.git](https://github.com/Devansh2835/chat-app.git).

## Features

- **Real-Time Messaging**: Instant messaging between users with WebSocket support.
- **User Authentication**: Secure login and registration system.
- **AI Chat Integration**: Interact with an AI chatbot for enhanced user experience.
- **Group Chats**: Support for group conversations.
- **Responsive Design**: Mobile-friendly interface for chatting on the go.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (with frameworks like React if applicable)
- **Backend**: Node.js, Express.js
- **Real-Time Communication**: Socket.io
- **Database**: MongoDB for storing messages and user data
- **AI Integration**: Integration with AI services (e.g., OpenAI API)
- **Authentication**: JWT or similar for user sessions

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/averra.git
   cd averra/chat-app
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file with API keys for AI services, database connections, etc.

4. Start the application:
   ```
   npm start
   ```

5. Open your browser and navigate to the local server.

## Usage

- Register or log in to access the chat features.
- Start private or group chats with other users.
- Engage with the AI chatbot for assistance or entertainment.
- View message history and manage conversations.
